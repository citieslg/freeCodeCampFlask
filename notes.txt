Уже с установленным Python глобально.
1.Создать дирректурию с проектом и перейти в нее
2. В консоли создать виртуальную среду командой : python3 -m venv env
3. После установки я перешел по каталогу cd env/Scripts и запустил activate
что создало/активировало среду окружения с именем env
4.после написали код Flask hello world
запустили в консоли : python appl.py
Создался сервер с исполнением кода из appl.py
5.Загружаем страницу в браузере по порту 5000
 http://127.0.0.1:5000/
6. Создаем папки static templates, где будем хранить HTML CSS JS
7. В папке templates создаем index.html и переписываем Flask
c использованием render_template, что отображает наш index.html
8.в папке static создаем папку css и создаем main.css и с помощью Flask url_for 
подключаем через форму css
9. Импортируем модуль flask_sqlalchemy и создаем БД на базе sqlite
10. В консоли корневого каталога проекта запускаем python импортруем модуль нашего приложения appl.py объект db
from appl import db
db.create_all() и мы создали БД в корневом каталоге проекта test.db
11. После update main.css Не отображались новые стили. Это происходит из-за того что файлы стилей кэшируются у тебя в браузере. Отчистить кэш можно с помощью CTRL+F5. Можно отключить кэширование совсем app.config['SEND_FILE_MAX_AGE_DEFAULT'] = 0 или {{ url_for( 'static', filename='style.css', v=1)}} указывать версию
